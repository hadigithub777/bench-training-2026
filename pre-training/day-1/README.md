# Day 1 — Python Fundamentals

Basics: variables, data types, control flow, loops.

## Scripts

### exercise_1.py — Data Types + Operators
- Creates variables (string, int, bool, float) and prints a formatted summary.
- Calculates years until retirement and weekly coffee budget.

### exercise_2.py — Control Flow
- `grade_classifier(score)` returns Distinction / Pass / Fail.
- Loops through a list of scores and prints results in a table.

### exercise_3.py — Multiplication Table Generator
- Asks the user for a number between 1-12 and prints the table.
- Input validation with a while loop.
- Supports printing all tables at once with `all`.

## How to Run

```bash
python3 exercise_1.py
python3 exercise_2.py
python3 exercise_3.py
```
